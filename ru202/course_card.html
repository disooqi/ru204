What you will learn

Redis Straems are a new feature in Redis 5.0. In this course we will cover the basic concepts of streaming before we dive into the producer and consumer. We will explore consuming groups before diving into the various management aspects of streams and the internals.


During the course we will cover typical use cases and dive into code implementations using Streams.

What you will build

Various data models to support use case for the Redis Games application. The redis-cli will be used for interactive example, code solutions will be provided in Python.
