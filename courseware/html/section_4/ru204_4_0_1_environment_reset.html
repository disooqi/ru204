Before going further, let's clean up your environment a bit and reset it to a known state:

1. If you're running Redis Stack in Docker, make sure that the container is running. If you're using Redis Stack in the cloud, there's nothing to do for this step so go to step 2!

2. Connect to your Redis instance using redis-cli or RedisInsight. If you're using RedisInsight, open up the built-in CLI.

3. Enter the following command to delete all of the keys and search indexes in Redis:

FLUSHDB

Redis should respond with:

OK

4. Make sure that you have the latest version of the course repository files on your machine. Run the following command in your local ru204 folder:

git pull origin main

OK, you're ready to move forwards! If you have trouble resetting your environment, please contact us on Discord - we're here to help!
