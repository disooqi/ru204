You've reached section 4 of the course! Now it's time to see how you can integrate the JSON search capabilities that were introduced in section 3 into your application. We'll look at examples written in Python, .NET (C#), Node.js and Java. You'll learn how to query a collection of documents with the Redis OM clients.

Remember that to be successful on this course, you don't need to know all four of the programming languages. While we'd recommend that you read each unit that follows, feel free to focus on the language(s) that are relevant to your application development needs and personal experience.

It's also important to note that you don't have to use the Redis OM clients to leverage functionality from RediSearch in an application. The lower level clients redis-py (Python), Node-Redis (Node.js), Jedis (Java) all include support for RediSearch commands, and StackExchange.Redis (.NET) allows you to run these commands using the generic ExecuteAsync function to send arbitrary commands to Redis.
